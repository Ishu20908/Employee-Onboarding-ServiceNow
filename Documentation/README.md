# Employee Onboarding Automation System

## Project Overview
This ServiceNow project automates the employee onboarding process from employee request submission to account creation, asset allocation, and onboarding task completion.

## Objectives
- Automate onboarding workflow
- Reduce manual effort
- Improve employee experience
- Track onboarding progress

## Modules Used
- Service Catalog
- Flow Designer
- Workflow
- Notifications
- Reports & Dashboards

## Features
- New Employee Request Form
- Manager Approval
- HR Verification
- IT Account Creation
- Asset Allocation
- Welcome Email Notification
