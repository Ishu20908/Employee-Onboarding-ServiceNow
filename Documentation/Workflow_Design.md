# Workflow Design

## Workflow Steps

1. HR submits onboarding request.
2. Manager receives approval request.
3. Manager approves onboarding request.
4. HR onboarding tasks are created.
5. IT onboarding tasks are created.
6. User account is created.
7. Laptop and required assets are allocated.
8. Welcome email is sent.
9. All tasks are completed.
10. Request is closed.
